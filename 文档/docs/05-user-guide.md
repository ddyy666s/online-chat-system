## 第1章 系统概述

### 1.1 系统简介

WebChat 是一款面向企业用户的在线即时通讯系统，提供安全、稳定、高效的实时沟通服务。系统支持以下核心功能：

| 功能分类 | 功能项 |
|----------|--------|
| 即时消息 | 文本消息、图片消息、文件传输、语音消息 |
| 实时通信 | 一对一语音通话、视频通话 |
| 社交管理 | 好友添加、分组管理、群组管理 |
| 个性化 | 个人资料编辑、自定义表情、好友印象 |
| 管理能力 | 用户管理、消息审计、系统通知、数据统计 |

### 1.2 运行环境要求

**客户端**:

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| 操作系统 | Windows 10 / macOS 10.15 / Ubuntu 20.04 | Windows 11 / macOS 13+ |
| 浏览器 | Chrome 90+ / Firefox 90+ / Edge 90+ | Chrome 最新版 |
| 屏幕分辨率 | 1280 x 720 | 1920 x 1080+ |
| 网络带宽 | 2 Mbps | 10 Mbps+ |
| 麦克风/摄像头 | 可选（用于音视频通话） | 推荐 USB 麦克风 |

### 1.3 快速访问

| 环境 | 访问地址 |
|------|----------|
| 生产环境 | `https://chat.example.com` |
| 测试环境 | `https://test.chat.example.com` |
| 开发环境 | `http://localhost:5173` |

### 2.2 注册账号

> **待补充截图**: 注册页面截图

**操作步骤**:

```
Step 1: 在登录页面点击"注册"链接，进入注册页面
Step 2: 填写以下信息：
    ┌─────────────────────────────────────────────┐
    │ 用户名: [________________] 4-50位字母/数字   │
    │ 密  码: [________________] 8-50位,需含字母数字│
```

- 密码强度要求：至少 8 位，需同时包含字母和数字
- 昵称可随时修改

## 第4章 聊天功能

### 4.1 发起私聊

**操作步骤**:

```
Step 1: 在左侧"好友"标签页中点击好友
Step 2: 右侧打开聊天窗口
Step 3: 在底部输入区域输入消息
Step 4: 按 Enter 键或点击"发送"按钮
```

> **待补充截图**: 私聊窗口截图

### 4.2 消息类型

| 消息类型 | 发送方式 | 说明 |
|----------|----------|------|
| **文本消息** | 直接在输入框输入，Enter 发送 | 支持 Shift+Enter 换行 |
| **图片消息** | 点击工具栏图片图标，选择图片 | 支持 jpg/png/gif/webp，≤ 10MB |
| **文件消息** | 点击工具栏文件图标，选择文件 | 任意格式，≤ 50MB |
| **语音消息** | 点击语音图标，按住录制，松手发送 | ≤ 60 秒，amr 格式 |

> **待补充截图**: 工具栏各按钮截图

#### 文本消息

```
[输入框]
  Shift+Enter = 换行
  Enter      = 发送
```

#### 图片消息

```
Step 1: 点击工具栏中的 📷 图标
Step 2: 在弹出的文件选择器中选择图片
Step 3: 图片上传中...（显示进度）
Step 4: 上传成功后自动发送
```

#### 语音消息

> **待补充截图**: 语音录制界面截图

```
Step 1: 点击输入框旁边的 🎤 图标
Step 2: 界面上出现录音按钮

         ┌─────────────┐
         │   按住说话    │
         └─────────────┘

Step 3: 按住按钮开始录制（显示录音时长）
Step 4: 松手发送语音消息
Step 5: 上滑到"取消"区域松手可取消发送
```

### 4.3 消息撤回

**操作步骤**:

```
Step 1: 鼠标悬停在自己发送的消息上
Step 2: 出现操作菜单，点击"撤回"按钮
         ┌──────────────────────┐
         │ 这是一条已发送的消息   │
         │         [撤回] [复制] │
         └──────────────────────┘
Step 3: 系统提示"你撤回了一条消息"
Step 4: 对方看到"对方撤回了一条消息"
```

> **待补充截图**: 消息撤回操作截图

**限制条件**:
- 仅可撤回 2 分钟内发送的消息
- 仅可撤回自己发送的消息
- 撤回后双方均不可查看原始内容

### 4.4 聊天历史

**使用方式**:

```
Step 1: 打开聊天窗口后，默认显示最新 20 条消息
Step 2: 向上滚动到消息列表顶部
Step 3: 自动触发加载更多历史消息（每次 20 条）
Step 4: 持续滚动可加载全部历史
```

> **待补充截图**: 滚动加载历史截图

## 第13章 常见问题

### 13.1 登录问题

**Q: 无法登录，提示"用户不存在"**

可能原因：
1. 用户名输入错误（区分大小写）
2. 账号尚未注册

解决方法：
- 检查用户名拼写是否正确
- 确认是否已注册，如未注册请先注册

**Q: 无法登录，提示"密码错误"**

可能原因：
1. 密码输入错误
2. 大小写锁定开启
3. 密码被修改

解决方法：
- 检查 Caps Lock 是否开启
- 重新输入密码
- 联系管理员重置密码

**Q: 无法登录，提示"用户已被禁用"**

原因：账号被管理员禁用

解决方法：
- 联系系统管理员申请解禁
- 等待管理员处理

### 13.2 消息问题

**Q: 消息发送失败**

| 现象 | 原因 | 解决方法 |
|------|------|----------|
| 消息旁边出现 ⚠️ 图标 | 网络连接断开 | 检查网络后重试 |
| 消息发送后无响应 | WebSocket 断开 | 刷新页面重新连接 |
| 图片上传失败 | 文件过大 | 压缩图片后重试 (≤ 10MB) |
| 语音发送失败 | 录制时间过短 | 录制时间需 ≥ 1 秒 |

**Q: 看不到对方的消息**

| 现象 | 说明 |
|------|------|
| 显示"对方撤回了一条消息" | 对方已撤回该消息 |
| 显示空白消息 | 消息类型不支持（极少见） |
| 消息显示加载中 | 图片/语音正在加载 |

**Q: 撤回消息失败**

```
错误提示: "消息已超过撤回时间"
解决: 消息发送超过 2 分钟不可撤回

错误提示: "消息已撤回"
解决: 该消息已被其他人或您自己撤回
```

### 13.3 音视频通话问题

**Q: 无法发起通话**

| 原因 | 解决方法 |
|------|----------|
| 对方不在线 | 文字留言，等待对方上线 |
| 浏览器权限未授予 | 在浏览器设置中允许使用麦克风/摄像头 |
| RTC Token 获取失败 | 刷新页面重试 |

**Q: 通话建立失败**

```
Step 1: 检查网络连接
Step 2: 刷新页面重新连接
Step 3: 检查防火墙设置
Step 4: 尝试切换为语音通话
```

**Q: 通话质量差**

| 问题 | 建议 |
|------|------|
| 画面模糊 | 确保网络带宽 ≥ 2 Mbps |
| 声音断断续续 | 关闭摄像头，仅用语音 |
| 回声严重 | 佩戴耳机，降低扬声器音量 |
| 延迟高 | 关闭其他占用带宽的应用 |

**Q: 浏览器权限设置**

```
Chrome 设置方法:
设置 → 隐私和安全 → 网站设置 → 麦克风/摄像头
→ 允许 chat.example.com 访问

Edge 设置方法:
设置 → Cookie 和网站权限 → 麦克风/摄像头
→ 允许 chat.example.com 访问
```

### 13.4 好友与群组问题

**Q: 搜索不到用户**

| 可能原因 | 解决 |
|----------|------|
| 用户名拼写错误 | 确认正确拼写 |
| 搜索关键词太短 | 关键词不少于 2 个字符 |
| 对方已注销 | 无法搜索到已注销用户 |
| 自己已被对方拉黑 | 系统未实现拉黑功能(如有) |

**Q: 好友请求相关问题**

| 错误提示 | 含义 |
|----------|------|
| "已经是好友" | 对方已在您的好友列表中 |
| "请求已发送，请等待处理" | 您已发送过请求，请勿重复发送 |
| "请求已过期" | 请求超过 7 天未处理，重新发送 |

**Q: 群组管理问题**

| 问题 | 解决方法 |
|------|----------|
| 无法邀请成员 | 确认对方是您的好友 |
| 无法设置管理员 | 仅群主可操作 |
| 无法解散群组 | 仅群主可操作，或先退出（群主不可退出）|
| 被移出了群组 | 可重新被邀请 |

### 13.5 系统与浏览器问题

**Q: 页面显示异常**

```
强制刷新: Ctrl + F5 (Windows) / Cmd + Shift + R (Mac)
清除缓存: 浏览器设置 → 清除浏览数据 → 缓存图片和文件
检查版本: 确认浏览器为最新版本
```

**Q: 某些功能无法使用**

| 检查项 | 操作 |
|--------|------|
| 浏览器版本 | 升级至 Chrome 90+ / Edge 90+ |
| JavaScript | 确认已启用 |
| Cookie | 确认已启用 |
| 浏览器插件 | 尝试禁用广告拦截等插件 |

**Q: 长时间无操作后被断开**

| 现象 | 说明 |
|------|------|
| 页面显示"连接断开" | WebSocket 因超时断开 |
| 消息发送失败 | 需要重新连接 |

解决方法：
- 刷新页面自动重连
- Token 未过期时可正常恢复
